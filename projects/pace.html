PACE

A mobile running coach app with readiness scoring, AI coach agents, backend sync, and public-launch preparation.

GitHub repository Back to projects

What I built

PACE is a running-focused coaching app. It turns daily check-ins into readiness scores, training notes, recent warning flags, recovery guidance, and a logged plan for the day.

The app includes four coach agents: Head Coach, Training Coach, Recovery Coach, and Injury Manager.

Current status

  • Expo React Native mobile app
  • Express backend with account and synced app data
  • SQLite storage for local development
  • OpenAI-backed coach chat proxy
  • Medical disclaimer and AI safety boundaries

Backend and launch work

The backend stores runner profiles, morning check-ins, coach chat messages, injury records, training plans, subscription status, and daily usage events. Public-launch preparation includes production environment templates, account recovery, data export, delete account, rate limits, deployment files, EAS build configuration, RevenueCat billing scaffolding, and draft privacy, terms, and safety documents.
