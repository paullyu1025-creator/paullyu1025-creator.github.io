GitHub repository Back to projects

What I built

I built an autonomous pacing robot that gives runners a visual pace reference while also analyzing running posture in real time. The system combines a three-wheel chassis, dual cameras, embedded AI inference, and motor control.

A front camera detects track lines for autonomous steering, while a side camera captures running form for posture analysis and feedback.

Technical work

  • YOLO-style trackline segmentation for outdoor running tracks
  • MediaPipe pose estimation for running form analysis
  • RK3588 embedded platform for AI inference
  • Arduino Nano microcontrollers for servo and speed control
  • PID control and encoder feedback for speed stability

Results

Speed control1.96% average speed error
