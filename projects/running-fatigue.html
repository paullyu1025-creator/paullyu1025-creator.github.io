Running Fatigue Detection

A Pioneer project under development using wearable sensors to study fatigue-related running-form changes.

GitHub repository Back to projects

Project idea

This project asks whether wearable sensors, including accelerometers and gyroscopes, can collect running-motion data that reveals fatigue-related posture and gait deviations.

Instead of judging every runner against one fixed ideal form, the project compares motion against each runner's own low-fatigue baseline.

Planned work

  • Review wearable-sensor gait research and datasets
  • Choose sensor placement and hardware
  • Filter raw IMU data into usable motion signals
  • Analyze cadence, trunk movement, stride length, and symmetry
  • Compare fatigue-state motion against a personal baseline

Status

This project is still under development. The current material is a research proposal and early planning around datasets, sensors, and validation.
